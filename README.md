# AI Governance Readiness Assessment

An interactive governance readiness assessment tool built for enterprise AI deployments.

20 questions across five governance domains:
- Data Governance
- Semantic Layer
- Model & AI Controls
- Observability
- Governance Operations

Built with React + Vite. Deploys to Vercel with zero configuration.

## Development

```bash
npm install
npm run dev
```

## Deploy

Push to GitHub. Connect repo to Vercel. Done.
